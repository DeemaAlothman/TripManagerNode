# TripManagerNode

**TripManagerNode** هو مشروع لإدارة الرحلات (Trips) مبني باستخدام **Node.js** و **Prisma ORM**.  
يوفر API بسيط لإضافة، تعديل، حذف، واستعراض الرحلات.

---

## 🛠️ المتطلبات

- Node.js (v16 أو أحدث)  
- NPM أو Yarn  
- قاعدة بيانات (MySQL أو PostgreSQL أو SQLite)  
- Prisma
